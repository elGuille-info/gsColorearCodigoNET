# gsColorearCodigoNET
Utilidad (de .NET 5.0 RC1) para colorear código y generar HTML para usar en sitios WEB.<br>
<br>
<br>
Esta utilidad utiliza .NET 5.0 RC1 y la biblioteca de clases <b>gsColorearNET</b> para colorear el código (de .NET Standard 2.0) instalada desde el paquete de NuGet.<br>
<br>
La carpeta <b>rc1</b> (antes preview)tiene el código del proyecto actual con las modificaciones (aún utiliza .NET 5.0 RC1 y está codificado con Visual Studio 2019 Preview Version 16.8.0 Preview 3).<br>
Las modificaciones estarán en otras ramas, por eso en este proyecto no uso otra carpeta con las diferentes versiones del código.<br>
<br>
La carpeta <b>bin-RC1</b> (antes bin-preview) tiene el ejecutable compilado con .NET 5.0 RC1 y gsColorearNET.dll compilada con .NET Standard 2.0.<br>
Con fecha del 19 de septiembre he añadido la carpeta <b>Resources</b>
<br>
<br>
<h3>Nota sobre esta versión para .NET 5.0 RC1</h3>
Esta versión para .NET 5.0 RC1 de la utilidad de gsColorearCodigo para .NET Framework 4.8 es para poder depurar la DLL gsColorearNET,<br>
ya que al estar compilada para .NET Standard 2.0 se ve que la depuración no la hace bien en Visual Studio 2019 final y sí en la versión preview, 
o al menos esa es lo que yo he podido comprobar.
