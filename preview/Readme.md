En esta carpeta está el código fuente de la utilidad para usar con Visual Studio 2019 Preview Version 16.8.0 Preview 2.1 (o posterior)<br>
<br>
Este proyecto utiliza el proyecto de la utilidad gsColorearNET el cual puedes descargar desde <a href="https://github.com/elGuille-info/gsColorearNET">aquí</a>.<br>
También utiliza la utilidad ColorearCore que está compilada para .NET Core 3.1 y que pronto publicaré (con el nombre gsCompilarCore). El código es en C#.<br>
<br>
En esta primera actualización (para la versión 1.0.04 de la utilidad con la versión 1.0.0.6 de gsColorearNET) 
no está el fichero de solución de Visual Studio, solo el del proyecto de Visual Basic .NET.<br>
<br>
Puede que en posteriores actualizaciones publique ese fichero de la solución e incluso el código de gsColorearNET, con idea de que 
puedas abrir la aplicación y la DLL en Visual Studio 2019 Preview.<br>
<br>
Lo que no estará es el fichero de firma elGuille.snk, que utilizan la DLL y posiblemente usará también la utilidad.<br>
